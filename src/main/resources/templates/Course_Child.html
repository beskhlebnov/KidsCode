Тема 1: Синтаксис Python и первая программа

Синтаксис Python

Синтаксис Python — это набор правил и соглашений, которые определяют, как должен быть написан код Python, чтобы он правильно интерпретировался интерпретатором Python. Он определяет структуру и форматирование программ Python. Код Python записывается в виде последовательности операторов. Каждый оператор представляет собой действие или команду. Например, присвоение значения переменной, вызов функции или определение цикла — все это примеры операторов. Кроме того, Python использует отступы для определения блоков кода. В отличие от других языков программирования, которые используют фигурные скобки или ключевые слова, Python использует отступы для обозначения начала и конца блоков кода. Обычно для отступов используются четыре пробела или табуляция.


Первая программа

Программа «Привет, мир!» — это распространённая отправная точка для новичков в любом языке программирования. Она просто выводит на экран текст «Привет, мир!». В Python это можно сделать с помощью всего одной строки кода.

print("Привет, Мир!")

Тема 2: Переменные Python

Если нам нужно что-то сохранить в реальной жизни, нам нужен контейнер или коробка для хранения. Переменные — это не что иное, как простые контейнеры для хранения данных разных типов, таких как числа, строки, логические значения, символы, списки, кортежи, диапазоны и т. д.

# Допустим, у вас есть коробка для хранения вещей.
# Вы можете дать коробке имя, например "my_box".
# Здесь "my_box" - это переменная.
my_box = "toys"
print(my_box) # Вывод: игрушки
# Переменные также могут хранить различные типы данных.
# Например, вы можете хранить числа в переменной.
my_number = 10
print(my_number) # Вывод: 10
# Вы также можете комбинировать переменные вместе.
my_message = "Привет, " + "Мир!"
print(my_message) # Вывод: Привет, мир!

Правила для переменных Python:

Вот некоторые из упомянутых правил, которые необходимо учитывать при выборе имени для переменной Python:

- Имя переменной в Python должно начинаться с буквы или символа подчеркивания.
- Имя переменной Python не может начинаться с числа.
- Имя переменной в Python может содержать только буквенно-цифровые символы и подчёркивания (A-z, 0-9 и _).
- Имена переменных в Python чувствительны к регистру (name, Name и NAME — это три разные переменные).
- Зарезервированные слова (ключевые слова) в Python нельзя использовать в качестве имён переменных в Python.

Тема 3: Типы данных Python

Типы данных в Python похожи на разные виды коробок, в которых можно хранить разные вещи. Если у вас есть разные виды игрушек или книг, то в Python есть разные типы данных, с которыми он может работать.


1. Целое число (int):

Целое число представляет собой целые числа без десятичных знаков. Оно похоже на целое печенье и представляет собой полное число без частей или дробе

Например, когда кто-то спрашивает нас о возрасте, мы обычно называем число без десятичных знаков, например, 18, 21, 11 и т. д.


age = 10
print(age) #Вывод 10

2. Число с плавающей точкой (float):

Число с плавающей точкой представляет собой числа с десятичными знаками. Это как печенье, в котором не хватает нескольких кусочков. Оно представляет собой число с десятичной или дробной частью. Его можно использовать для представления измерений.

Например, нам нужно измерить высоту, поэтому измерение должно быть кратким и точным, оно включает в себя число + дробную часть, например, 5,5, 6,7, 7 и т. д.


height = 5.5
print(height) #Вывод 5.5

3. Строка (str):

Строка представляет собой последовательность символов или букв. Она может использоваться для хранения текстовой информации. Строка похожа на набор букв, слов или предложений.

Например, имя человека может храниться в виде строки.


name = "Игорь"
print(name) #Вывод Игорь

4. Логическое значение (bool):

Булево значение представляет собой два возможных варианта: «истина» или «ложь». Оно может принимать только значения «истина» или «ложь». Булево значение похоже на переключатель, который может быть включён или выключен. Оно представляет собой значение «истина» или «ложь», как ответ «да» или «нет» на вопрос.

Например, является ли Москва столицей Росии или нет? Таким образом, возможный ответ на этот тип вопроса — «да» или «нет», но в компьютерах мы используем понятия «истина» (Ture) и «ложь» (False).


is_Moscow_Capital_of_Russia = True #Является ли Москва столицей Росии
print(is_Moscow_Capital_of_Russia) #Вывод Ture(То есть истина, или правда)

5. Список (List):

Список — это упорядоченная коллекция элементов. Его можно использовать для хранения нескольких значений разных типов данных. Список похож на корзину с товарами. Он может содержать разные элементы, например числа, слова или даже другие списки.

Пример 1: Список покупок может содержать элементы разных типов, например строки, числа или логические значения.


fruits = ["Яблоко", "Банан", "Апельсин"]
print(fruits) #Вывод ['Яблоко', 'Банан', 'Апельсин']

Давайте разберемся с типом данных List на еще одном примере:

Пример 2: представьте, что у вас есть список ваших любимых цветов. Вы можете записать их один за другим.


colors = ["Зелёный", "Чёрный", "Желтый"]
print(colors) #Вывод ["Зелёный", "Чёрный", "Желтый"]

Вы можете легко добавлять или удалять цвета из своего списка в любое время. Списки похожи на коллекции элементов, которые вы можете легко изменять или обновлять, как и свой список любимых цветов.
